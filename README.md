##cocos 2d 3.X游戏开发《2048》
工具使用：`VS2012`,语言使用`C++`

2048游戏很普及，此模仿原版2048开发的游戏。

在我开发过程中此游戏难点是**手势的识别**，获取前后坐标需要判断方向，纵坐标容易搞反啊= =

已有项目包可直接copy 代码`Classes`和资源`Resource`文件夹到项目文件夹下运行
